## 0. A both-owed directional dichotomy (two specs)

Scalping's micro edge could be **mean-reversion** (fade the last bar) OR **momentum** (chase it). Per the handoff, **P3.13 Scalping MR/momentum is a genuine both-owed dichotomy** — so BOTH are drafted and owed: `scalp_mean_reversion` and `scalp_momentum`. They are **exact opposites** — on every triggered bar they take opposite sides — so divergence is **definitional** (not the subtle P3.7 KAMA case; a divergence test asserts it directly, `test_mr_and_momentum_take_exact_opposite_sides`).

## 1. Hypothesis (falsifiable) — per spec

On the frozen large-cap **panel** at 5-min bars, fast micro fade / chase produces a **positive net-of-cost edge** clearing the **two-part panel verdict**, net of the full round-trip cost. **Falsified if** the panel verdict is KILL/INSUFFICIENT, or (expected) the §6 pre-check finds the median gross per-trade move below cost.

## 2. Economic rationale (mechanism, not pattern) — PURELY a-priori

- **MR:** at the finest scale, order-flow noise overshoots and **snaps back** — fade the last bar's move.
- **Momentum:** at the finest scale, a burst of one-sided flow **persists for a bar or two** — chase it.

Both are micro-structure bets, **extremely cost-sensitive** (tiny per-trade edge vs a fixed round-trip cost). Purely a-priori.

## 3. Specs (event → entry → exit/holding → position)

In [`src/lab/research/strategies/scalping.py`](../../src/lab/research/strategies/scalping.py). Per bar, from the **same-day** last-bar return `close[i]/close[i-1]-1` (the day's first bar has no valid prior-bar return ⇒ flat, so an overnight gap is never scalped):

- **`ScalpMeanReversionSpec`** (`scalp_mean_reversion`): `ret > +threshold` ⇒ SHORT, `< −threshold` ⇒ LONG.
- **`ScalpMomentumSpec`** (`scalp_momentum`): `ret > +threshold` ⇒ LONG, `< −threshold` ⇒ SHORT.
- Held one bar (the next bar's target overrides), so **turnover is very high**.

**Causal / no-lookahead (hard precondition).** The per-bar target uses only bars `≤ i`, fills at `i+1`'s open. **Prefix-invariance no-lookahead test for both specs** (`tests/unit/test_scalping_spec.py::test_scalp_prefix_invariance_both_specs`) is a **hard precondition** — the study does not run if it fails. Confirmed **PASSED**.

**FREQUENCY CAVEAT (flagged).** Scalping's natural bars are **1–3 min**; the archive holds **5-minute bars only**, so this is tested at 5-min — a **coarser-than-natural proxy**. Per the catalog, P3.13 is included to **demonstrate the cost wall honestly**; cost-death at 5-min is the expected §6 landing, and a 1–3-min re-test would require backfilling finer data (deferred, operator's call).
