## 2. Economic rationale (mechanism, not pattern) — PURELY a-priori

Two exhaustion signals **agreeing** should improve the fade: a band touch (statistical stretch) that is *also* an RSI extreme (momentum exhaustion) is likelier a genuine overshoot than either alone. Purely a-priori; honest caution — both legs **fail together in strong trends** (fading a real move), and it is a common textbook combo that rarely survives honest OOS + costs.

## 3. Spec (event → entry → exit/holding → position)

In [`src/lab/research/strategies/bollinger_rsi.py`](../../src/lab/research/strategies/bollinger_rsi.py). `BollingerRsiSpec`, symmetric, per-bar with hysteresis: ENTER LONG when `close < lower band` AND `RSI < rsi_oversold`; SHORT when `close > upper band` AND `RSI > 100 - rsi_oversold`. EXIT on reversion to the Bollinger **middle band**. Flat each day; square-off.

**Causal / no-lookahead (hard precondition).** `talib` Bollinger + RSI are trailing (prefix-invariant); the RSI leg limits gap-driven entries. **Prefix-invariance test** (`test_bollinger_rsi_spec.py::test_bollinger_rsi_prefix_invariance`) = hard precondition, **PASSED**.
